# Happiness Mod

A Minecraft mod that adds a happiness/mood system. Your character's happiness drains in darkness and restores in sunlight.

## Features

- **Happiness Meter**: Drains slowly in darkness, restores in light
- **Mood Effects**: Get speed/haste when euphoric, slowness/mining fatigue when sad
- **Happiness Ore**: New ore that spawns like coal, mine it for Raw Happiness
- **Happiness Pills**: Smelt raw happiness into pills, eat them to restore mood
- **HUD Display**: See your current happiness level on screen

## Mood Levels

| Level | Happiness | Effects |
|-------|-----------|---------|
| Euphoric | 90+ | Speed + Haste |
| Normal | 30-90 | No effects |
| Sad | 10-30 | Slowness + Mining Fatigue |
| Depressed | 0-10 | Slowness II, Mining Fatigue II, Weakness |

## Requirements

- Minecraft 1.21.1
- Fabric Loader
- Fabric API
- Architectury

## Installation

1. Install Fabric Loader for Minecraft 1.21.1
2. Download and install Fabric API
3. Download and install Architectury
4. Download this mod and place in your mods folder
